HTML

HTML är ett markerings-språk som används mest för att skapa websidor. Det används genom "markeringar" eller element för att strukturera up och delvis ändra form på ett dokument. HTML låter också en att inkludera andra filer i dokumentet, t.ex bilder, program (JavaScript) och intruktioner hur dokumentet ska visas (CSS).

CSS

CSS är ett stilmalls-språk som ändrar hur ett HTML dokument visas, med typsnitt, färg, storlek, etc.

JavaScript

JavaScript är ett scriptspråk som oftast används på klientsidan av en hemsida för att skapa interaktivitet.
